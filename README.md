# Arlang
Arlang is an object-oriented programming language designed for simplicity and efficiency.

Syntax is really simple and similar to Python.

Compiler is GCC, but you can install it on the repos.

Main commands is:
'
print(Hello World!)
set(x,10)
calc(1+1)
random(10000)
'
**This is beta of Arlang, created by 13 y.o. This is my first project so pls dont hate me if i did something wrong.**

# INSTALL

Arlang does not need to be installed, because it is presented in the archive, but you can install GCC if for some reason it is not installed. And also open the file setuparlangforcmd.exe to install the command ''Arlang'' in cmd
